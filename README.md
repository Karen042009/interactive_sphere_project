# Գունդ և նրա մասերը - Ինտերակտիվ կրթական կայք

## Նախագիծի նկարագրություն

Այս նախագիծը ներկայացնում է գնդային երկրաչափությունը՝ ինտերակտիվ 3D մոդելների և մանրամասն տեսական նյութերի միջոցով։ 

## Տեխնոլոգիաներ

- **HTML5** - Կայքի կառուցվածք
- **CSS3** - Ոճավորում և անիմացիաներ
- **JavaScript** - Ինտերակտիվություն և տրանսլացիա
- **Three.js** - 3D գրաֆիկա և մոդելավորում
- **MathJax** - Մաթեմատիկական բանաձևերի ներկայացում

## Ինչպես օգտագործել

### Տարբերակ 1: Ուղղակի բացում (համեստ բրաուզերներում)

Պարզապես բացեք `index.html` ֆայլը ձեր բրաուզերում։

**Նշում:** Որոշ բրաուզերներ (օրինակ՝ Chrome) կարող են արգելափակել local JavaScript ֆայլերի բեռնումը անվտանգության պատճառներով։ Եթե առաջանան խնդիրներ, օգտագործեք Տարբերակ 2-ը։

### Տարբերակ 2: Պարզ HTTP սերվեր (Առաջարկվող)

#### Python-ով:
```bash
# Python 3
python -m http.server 8000

# Այնուհետև բացեք:
# http://localhost:8000
```

#### Node.js-ով (եթե տեղադրված է):
```bash
npx http-server -p 8000

# Այնուհետև բացեք:
# http://localhost:8000
```

### Տարբերակ 3: Live Server (VS Code)

Եթե օգտագործում եք Visual Studio Code:
1. Տեղադրեք "Live Server" ընդլայնումը
2. Աջ կտտացրեք `index.html` ֆայլի վրա
3. Ընտրեք "Open with Live Server"

## Պանակների կառուցվածք

```
interactive_sphere_project/
├── index.html              # Գլխավոր տեսական էջ
├── interactive.html        # Ինտերակտիվ 3D էջ
├── README.md              # Այս ֆայլը
└── static/
    ├── css/
    │   └── style.css      # Կայքի բոլոր ոճերը
    └── js/
        ├── main.js        # Ինտերակտիվ էջի տրամաբանություն
        ├── translations.js # Բազմալեզու աջակցություն (ՀԱՅ/ENG/ՌՈՒՍmaterial)
        ├── three.min.js   # Three.js գրադարան
        └── OrbitControls.js # Տեսախցիկի կառավարում
```

## Առանձնահատկություններ

✅ **Ամբողջությամբ static** - Առանց backend, աշխատում է ցանկացած վեբ սերվերի վրա  
✅ **Բազմալեզու** - Հայերեն, Անգլերեն, Ռուսերեն  
✅ **3D Ինտերակտիվ մոդելներ** - Պտույտ, zoom, գունային փոփոխություններ  
✅ **Ադապտիվ դիզայն** - Աշխատում է բոլոր սարքերում  
✅ **MathJax** - Մաթեմատիկական բանաձևերի գեղեցիկ ներկայացում

## Նախկին տարբերակ

Այս նախագիծը նախկինում օգտագործում էր Python Flask backend։ Այժմ այն ամբողջությամբ կախված չէ server-side տեխնոլոգիաներից և կարող է տեղակայվել ցանկացած static hosting-ի վրա (GitHub Pages, Netlify, Vercel և այլն)։
